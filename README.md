# What is Hacktoberfest?

Hacktoberfest is a month-long celebration of open source projects, their maintainers, and the entire community of contributors organized by Digital Ocean, DEV and Github, where you can receive a T-Shirt by making 4 valid pull requests in the month of October to open source projects on Github.

#### Register for Hacktoberfest : https://hacktoberfest.com/

## Steps to follow

1. Fork the repo and clone it
2. Create new branch
3. Do the required changes and commit them
4. Push your changes and branch to your forked repo
5. Open the pull request

The changes will be reviewed and if it is valid, merged into master.
